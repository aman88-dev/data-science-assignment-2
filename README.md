## Project Overview

This project focuses on analyzing movie-related data using Python and data analysis libraries.  
The aim is to identify patterns, trends, and insights related to movie performance, profitability, and key contributors such as actors, producers, and directors.

The analysis is implemented using a Jupyter Notebook.

---

## Tools and Technologies

- Python
- Jupyter Notebook
- Pandas – Data manipulation and analysis
- Matplotlib – Data visualization

---

## Dataset Description

The dataset contains information about various movies including:

- Profit
- Return on Investment (ROI)
- Language
- Genres
- Producers
- Actors
- Directors

This information is used to perform analytical queries and visualizations.

---

## Project Tasks

### 1. Top Movies by Profit
Identify the movies with the highest profit and visualize the results.

### 2. Average ROI by Language
Calculate the average Return on Investment (ROI) for movies based on language.

### 3. Genre Analysis
Determine the most frequent movie genres in the dataset.

### 4. Producer Analysis
Extract producer names and identify the top producers with the most movies.

### 5. Actor Appearance Analysis
Find which actors appear most frequently across movies.

### 6. Director Performance Analysis
Identify top directors based on average ROI to understand who creates the most profitable films.
